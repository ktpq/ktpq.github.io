เครื่องดื่ม - ร้อน

1. Espresso

กาแฟเข้มข้นที่ได้จากการสกัดแบบช็อต ให้รสชาติที่เข้มเต็มรสกาแฟ เหมาะสำหรับคนรักกาแฟที่ต้องการรสชาติเข้มข้น

ราคา : 50 บาท

วัตถุดิบ : กาแฟบดเข้ม, น้ำ

2. Americano

กาแฟดำที่มีการเติมน้ำร้อน เพิ่มรสชาติกลมกล่อม เหมาะสำหรับผู้ที่ชื่นชอบกาแฟรสชาติเรียบง่าย

ราคา : 60 บาท

วัตถุดิบ : กาแฟบดเข้ม, น้ำร้อน

3. Latte

กาแฟผสมนมร้อน รสชาติกลมกล่อม เนื้อเนียนนุ่ม ดื่มง่ายและให้ความรู้สึกผ่อนคลาย

ราคา : 70 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด

4. Cappuccino

กาแฟนมร้อนที่มีฟองนมหนานุ่ม ให้สัมผัสที่นุ่มละมุน พร้อมกลิ่นหอมของกาแฟ

ราคา : 70 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, ฟองนม

5. Mocha

กาแฟผสมช็อกโกแลตที่ให้รสหวานมัน ความลงตัวของช็อกโกแลตและกาแฟที่ไม่เหมือนใคร

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, ช็อกโกแลต, นมสด

6. Caramel Macchiato

กาแฟนมร้อนที่ราดด้วยซอสคาราเมล เพิ่มความหวานละมุนและความหอมของกาแฟ

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, ซอสคาราเมล

7. Flat White

กาแฟนมร้อนที่มีความเข้มข้นจากกาแฟเอสเปรสโซ่ และเนื้อสัมผัสที่นุ่มนวลจากฟองนม

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, ฟองนม

8. Honey Latte

กาแฟนมร้อนที่เติมความหวานจากน้ำผึ้งธรรมชาติ ให้ความหวานละมุนในทุกการดื่ม

ราคา : 70 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, น้ำผึ้ง

9. Macchiato

กาแฟเอสเปรสโซ่ที่ถูกผสมกับฟองนมให้ความเข้มข้นและเนียนนุ่มในทุกคำ

ราคา : 60 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, ฟองนม

10. Matcha Latte

มัทฉะลาเต้ที่ผสมกับนมร้อนอย่างลงตัว รสชาติหอมหวาน ละมุนทุกการดื่ม

ราคา : 80 บาท

วัตถุดิบ : มัทฉะ, นมสด

เครื่องดื่ม - เย็น

1. Iced Americano

กาแฟดำเย็นที่ให้รสชาติกลมกล่อมสดชื่น เหมาะกับการดื่มในทุกช่วงเวลา

ราคา : 60 บาท

วัตถุดิบ : กาแฟบดเข้ม, น้ำแข็ง

2. Iced Latte

กาแฟนมเย็นที่มีรสชาติกลมกล่อม เนื้อเนียนละมุนและทำให้รู้สึกสดชื่น

ราคา : 70 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, น้ำแข็ง

3. Iced Cappuccino

กาแฟนมเย็นพร้อมฟองนมที่เนียนนุ่ม ช่วยเติมความสดชื่นในทุกการดื่ม

ราคา : 70 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, ฟองนม, น้ำแข็ง

4. Iced Mocha

กาแฟเย็นผสมช็อกโกแลตหวานมัน หอมกลิ่นกาแฟและช็อกโกแลตเข้ากันอย่างดี

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, ช็อกโกแลต, นมสด, น้ำแข็ง

5. Iced Caramel Macchiato

กาแฟนมเย็นที่ราดซอสคาราเมลหวานหอม ละมุนในทุกการดื่ม

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, ซอสคาราเมล, น้ำแข็ง

6. Iced Matcha Latte

มัทฉะเย็นที่ผสมความหวานจากนมสด เติมความสดชื่นได้ในทุกเวลาของวัน

ราคา : 80 บาท

วัตถุดิบ : มัทฉะ, นมสด, น้ำแข็ง

7. Iced Cold Brew

กาแฟสกัดเย็นที่ให้รสชาติหอมกรุ่น ละมุนและไม่ขมจนเกินไป

ราคา : 90 บาท

วัตถุดิบ : กาแฟบดละเอียด, น้ำ, น้ำแข็ง

8. Iced Vanilla Latte

กาแฟนมเย็นที่มีกลิ่นหอมของวนิลา เพิ่มความหวานละมุนและสดชื่น

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, วนิลา, น้ำแข็ง

9. Iced Coconut Coffee

กาแฟเย็นผสมกะทิสด เติมความหอมหวาน และสัมผัสที่สดชื่น

ราคา : 90 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, กะทิสด, น้ำแข็ง

10. Iced Honey Latte

กาแฟนมเย็นที่หวานจากน้ำผึ้งธรรมชาติ เติมความสดชื่นและความหวานธรรมชาติ

ราคา : 80 บาท

วัตถุดิบ : กาแฟเอสเปรสโซ่, นมสด, น้ำผึ้ง, น้ำแข็ง

เครื่องดื่ม - น้ำผลไม้

1. Orange Juice

น้ำส้มสดคั้น 100% ที่มีรสชาติหวานและเปรี้ยวสดชื่น เหมาะสำหรับทุกช่วงเวลา

ราคา : 50 บาท

วัตถุดิบ : น้ำส้มสด

2. Mango Smoothie

สมูทตี้มะม่วงเนื้อเนียนนุ่ม หวานหอม รสชาติสดชื่นเหมาะกับวันร้อนๆ

ราคา : 80 บาท

วัตถุดิบ : มะม่วง, โยเกิร์ต, น้ำแข็ง

3. Watermelon Juice

น้ำแตงโมสดคั้นที่หวานชื่นใจ สดชื่นมากๆ สำหรับคลายร้อน

ราคา : 60 บาท

วัตถุดิบ : แตงโม

4. Pineapple Juice

น้ำสับปะรดคั้นสดที่มีรสชาติหวานอมเปรี้ยว สดชื่นและเต็มไปด้วยวิตามิน

ราคา : 55 บาท

วัตถุดิบ : สับปะรด

5. Lemonade

น้ำมะนาวสดผสมกับน้ำตาล เพิ่มความเปรี้ยวหวานสดชื่นในทุกการดื่ม

ราคา : 45 บาท

วัตถุดิบ : มะนาว, น้ำตาล

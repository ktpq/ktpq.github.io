ของทานเล่น - แซนต์วิช

1. Ham Cheese Sandwich

แซนวิชแฮมและชีสสุดคลาสสิก อบร้อนหอมกรุ่น

ราคา : 70 บาท

วัตถุดิบ : ขนมปัง, แฮม, ชีส, เนย

2. Egg Salad Sandwich

แซนวิชสลัดไข่ เนื้อนุ่ม รสชาติกลมกล่อม

ราคา : 65 บาท

วัตถุดิบ : ขนมปัง, ไข่, มายองเนส, ผักสด

3.Tuna Sandwich

แซนวิชทูน่าเข้มข้น เติมเต็มด้วยผักสด

ราคา : 75 บาท

วัตถุดิบ : ขนมปัง, ทูน่า, มายองเนส, ผักสด

4. Chicken Club Sandwich

แซนวิชไก่คลับ พร้อมเบคอนและผักสด รสชาติเข้มข้น

ราคา : 85 บาท

วัตถุดิบ : ขนมปัง, ไก่, เบคอน, ผักสด, มายองเนส

5. BLT Sandwich

แซนวิชเบคอน ผักกาดหอม และมะเขือเทศ หอมกรอบ

ราคา : 80 บาท

วัตถุดิบ : ขนมปัง, เบคอน, ผักกาดหอม, มะเขือเทศ

6. Grilled Cheese Sandwich

แซนวิชชีสย่าง กรอบนอกนุ่มใน อร่อยลงตัว

ราคา : 70 บาท

วัตถุดิบ : ขนมปัง, ชีส, เนย

1. Chicken Pie

ราคา : 60 บาท

วัตถุดิบ : แป้งพาย, ไก่, หอมใหญ่, มันฝรั่ง, เครื่องเทศ

2. Mushroom Puff

ราคา : 55 บาท

วัตถุดิบ : แป้งพัฟ, เห็ด, ครีม, ชีส

3. Apple Pie

วัตถุดิบ : แป้งพาย, แอปเปิล, น้ำตาล, เครื่องเทศ

วัตถุดิบ : แป้งพัฟ, ผักโขม, ชีส, เนย

5. Beef Pie

พายเนื้อรสชาติเข้มข้น เนื้อเนียนนุ่ม พร้อมเครื่องเทศหอมๆ

ราคา : 80 บาท

วัตถุดิบ : แป้งพาย, เนื้อบด, หอมใหญ่, เครื่องเทศ

วัตถุดิบ : แป้งพัฟ, สับปะรด, น้ำตาล, เนย

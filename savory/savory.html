<- อาหารคาว - สปาเกตตี้ ->

1. Spaghetti Carbonara

สปาเก็ตตี้คาโบนารา ครีมซอสหอมมันเข้มข้น พร้อมเบคอนกรอบ

ราคา : 120 บาท

วัตถุดิบ : สปาเก็ตตี้, ครีม, เบคอน, พาร์มีซาน

2. Spaghetti Bolognese

สปาเก็ตตี้ซอสโบโลเนส เนื้อเข้มข้น พร้อมเครื่องเทศหอมๆ

ราคา : 110 บาท

วัตถุดิบ : สปาเก็ตตี้, เนื้อบด, มะเขือเทศ, หอมแดง, กระเทียม, เครื่องเทศ

3. Spaghetti Aglio e Olio

สปาเก็ตตี้น้ำมันมะกอก รสเผ็ดเล็กน้อย หอมกระเทียม

ราคา : 100 บาท

วัตถุดิบ : สปาเก็ตตี้, น้ำมันมะกอก, กระเทียม, พริกแห้ง

4. Spaghetti Seafood

สปาเก็ตตี้ซีฟู้ด รสชาติจัดจ้าน พร้อมกุ้งและปลาหมึกสด

ราคา : 130 บาท

วัตถุดิบ : สปาเก็ตตี้, กุ้ง, ปลาหมึก, มะเขือเทศ, กระเทียม, พริก

<- อาหารคาว - อาหารเช้า ->

1. Classic American Breakfast

อาหารเช้าสไตล์อเมริกัน พร้อมไข่ดาว เบคอน และขนมปังปิ้ง

ราคา : 150 บาท

วัตถุดิบ : ไข่, เบคอน, ขนมปัง, เนย

2. Pancakes with Maple Syrup

แพนเค้กเนื้อนุ่ม เสิร์ฟพร้อมเมเปิลไซรัปและเนยสด

ราคา : 120 บาท

วัตถุดิบ : แป้ง, นม, ไข่, เมเปิลไซรัป, เนย

3. Avocado Toast

ขนมปังปิ้งหน้าอะโวคาโดบด โรยด้วยพริกป่นและมะนาว

ราคา : 130 บาท

วัตถุดิบ : ขนมปัง, อะโวคาโด, พริกป่น, มะนาว

4. Omelette with Veggies

ไข่เจียวสไตล์ฝรั่งเศส ใส่ผักสดและชีสรสกลมกล่อม

ราคา : 110 บาท

วัตถุดิบ : ไข่, ผักสด, ชีส
